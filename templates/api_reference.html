API Reference

REST API endpoints for SentinelAI security analysis

POST
/analyze

Analyze Solidity smart contract code for vulnerabilities.

Request Body

{
  "code": "pragma solidity ^0.8.0;
contract MyContract { ... }"
}

Response

{
  "score": 25,
  "summary": "CRITICAL: Contract has severe security flaws...",
  "findings_count": 5,
  "critical_count": 2,
  "high_count": 2,
  "medium_count": 1,
  "findings": [
    {
      "id": "reentrancy",
      "severity": "CRITICAL",
      "title": "Reentrancy",
      "description": "External call before state update...",
      "line": 15,
      "fix": "Update state variables BEFORE external calls...",
      "confidence": "92%"
    }
  ],
  "type": "contract"
}
POST
/scan-website

Scan a live website for security misconfigurations.

Request Body

{
  "url": "example.com"
}

Response

{
  "score": 45,
  "summary": "CRITICAL: example.com has severe security flaws...",
  "url": "https://example.com",
  "status": 200,
  "findings_count": 8,
  "critical_count": 1,
  "high_count": 4,
  "medium_count": 3,
  "findings": [
    {
      "id": "missing_https",
      "severity": "CRITICAL",
      "title": "Missing HTTPS Encryption",
      "description": "Website does not use HTTPS...",
      "fix": "Enable SSL/TLS certificate...",
      "impact": "Man-in-the-middle attacks...",
      "confidence": "100%",
      "location": "Protocol Layer"
    }
  ],
  "headers": { ... },
  "is_https": false,
  "type": "website"
}
