Everything you need to know about SentinelAI

Getting Started

Paste Solidity code or enter a website URL. Click "Run Security Audit" or "Scan Website" to get instant results.

Security Score

Scores range 0-100. Green (90+) is secure. Red (<50) means critical flaws. Fix issues in priority order.

Findings

Each vulnerability shows severity, description, line number, and fix. Toggle between "Vulnerable" and "Fixed" code tabs.

Executive Summary

After scanning, scroll down for a plain-English summary and step-by-step improvement roadmap.

Smart Contract Scanner

Detects 10+ Solidity vulnerabilities including Reentrancy, tx.origin, delegatecall, integer overflow, and more. Uses pattern matching to simulate AI analysis.

Website Scanner

Checks HTTPS, security headers (CSP, HSTS, X-Frame-Options), cookies, XSS vectors, CSRF tokens, and exposed sensitive files.

Limitations

This is a demo project for educational purposes. For production use, integrate with Slither, Mythril, OWASP ZAP, or hire professional auditors.
